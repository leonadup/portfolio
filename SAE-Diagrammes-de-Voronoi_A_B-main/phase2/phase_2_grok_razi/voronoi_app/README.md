# Application Diagramme de Voronoï - Phase 2

Application Python permettant de charger une liste de points depuis un fichier .txt, de calculer le diagramme de Voronoï (algorithme implémenté from scratch) et de le visualiser/exporter.

## Installation

```bash
python -m venv venv
```
# Windows
```bash
venv\Scripts\activate
```
# Linux/macOS
```bash
source venv/bin/activate
```

### 2. requirements.txt
```txt
matplotlib>=3.8.0
pytest>=8.0.0
```
```bash
pip install -r requirements.txt
```

## Utilisation

1. Lancez l’application :
   ```bash
   python -m src.main
   ```
