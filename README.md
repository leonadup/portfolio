# Portfolio - Léona DUPONT, troisième année en BUT Informatique

Ce site web est mon portfolio personnel, créé dans le cadre de mon BUT Informatique. Il présente mon profil, mon parcours, mes compétences, mes projets et mon carnet de voyage.

## 🧠 Contenu

- **Présentation** : Brève introduction sur mon parcours et ma recherche de stage.
- **Expériences & Formations** : Mon parcours de formation et mes expériences (dont le hackathon "Nuit de l'informatique").
- **Compétences** : Langages, bases de données, développement mobile, outils, méthodologies et langues.
- **Projets** : Sélection de projets réalisés pendant mes études ou en autonomie, avec des pages de détail dédiées.
- **Carnet de voyage** : Carte du monde interactive présentant les pays que j'ai visités.

## ✨ Fonctionnalités

- Mode sombre / clair
- Site multilingue (français, anglais, espagnol, coréen)
- Carte du monde interactive (survol pour plus de détails)
- Pages de détail pour chaque projet

## 🛠️ Technologies utilisées

- HTML
- CSS
- JavaScript
