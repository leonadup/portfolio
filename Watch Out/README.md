# Watch Out - Jeu d'Horreur

Un petit jeu pygame, d'exploration dans une forêt sombre.

## Comment lancer le jeu

### Prérequis

Il faut avoir Python 3.8+ d'installé. Après il faut installer pygame:

```bash
pip install pygame
```

### Lancer le jeu

Puis il faut lancer le jeu:

```bash
python jeu.py
```

## C'est quoi le jeu ?

On se retrouve perdu en forêt la nuit. Il y a 3 chiffres cachés qu'il faut trouver en explorant les scènes. Une fois tous les chiffres trouvés, il faut rentrer le code dans l'écran code avant que le temps expire.

## Les commandes

- **Souris** : Pour cliquer sur les zones cliquables et explorer
- **Chiffres** : Pour entrer le code
- **Backspace** : Pour annuler un chiffre
- **Entrée** : Pour valider le code
- **X en haut à droite** : Pour quitter l'écran de code
