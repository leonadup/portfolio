Implémentation d'un besoin client

Détection de communautés dans un réseau d'amitié, modélisé en Python à l'aide de dictionnaires.

Python Structures de données

Contexte

Cette SAÉ de première année consistait à répondre à un besoin client autour de l'analyse de réseaux sociaux : à partir d'une liste de relations d'amitié, déterminer les groupes de personnes qui se connaissent toutes mutuellement, appelés « communautés ».

Objectifs

Le projet, réalisé en binôme, visait à :

  • Modéliser un réseau d'amis sous forme de dictionnaire Python (chaque personne associée à la liste de ses amis)
  • Construire ce réseau automatiquement à partir d'une liste de paires d'amis
  • Identifier les communautés, c'est-à-dire les groupes où chaque membre est ami avec tous les autres membres du groupe
  • Valider le fonctionnement du programme à l'aide de tests unitaires

Réalisation

Nous avons développé une fonction create_network qui construit le réseau d'amitié sous forme de dictionnaire à partir d'une liste de paires (par exemple {"Alice": ["Bob", "Dominique"], ...}), puis un algorithme de détection de communautés qui parcourt ce réseau pour regrouper les personnes mutuellement connectées entre elles.

L'ensemble du code a été validé par une suite de tests unitaires (test_community_detection.py), couvrant différents cas de figure : réseaux vides, communautés isolées, ou personnes appartenant à plusieurs groupes.

Mon rôle

En binôme, j'ai travaillé sur la conception des structures de données (dictionnaires représentant le réseau), sur l'algorithme de détection des communautés et sur l'écriture des tests unitaires permettant de vérifier sa robustesse sur différents jeux de données.

Résultats

Le programme final identifie correctement les communautés d'un réseau d'amis fourni en entrée, et l'ensemble des tests unitaires passe avec succès. Ce projet a renforcé ma maîtrise des structures de données Python et des bonnes pratiques de test.
