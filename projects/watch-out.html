Watch Out

Petit jeu vidéo d'horreur en 2D développé en Python avec Pygame, dans le cadre d'un projet multimédia scolaire.

Python Pygame Travail individuel

Contexte

Ce projet a été réalisé dans le cadre d'un projet multimédia scolaire. L'objectif était de concevoir un petit jeu vidéo complet, de l'histoire jusqu'à la mécanique de jeu, en passant par les graphismes, les sons et l'interface.

Objectifs

L'objectif était de créer un jeu d'horreur en 2D capable de :

  • Faire explorer plusieurs scènes reliées entre elles (forêt, chemin, maison hantée)
  • Cacher des indices (chiffres) dans le décor que le joueur doit retrouver
  • Mettre en place un système de code à entrer avant la fin d'un minuteur
  • Créer une ambiance angoissante grâce aux graphismes, à la musique et à des apparitions surprises

Réalisation

J'ai développé le jeu avec Python et la bibliothèque Pygame. Le joueur explore plusieurs scènes liées par des zones cliquables, doit repérer 3 chiffres cachés dans le décor, puis les saisir sur un écran de code avant la fin d'un minuteur de 20 secondes. Pour renforcer l'ambiance, j'ai ajouté une musique d'ambiance, des fantômes semi-transparents qui se déplacent vers le joueur, ainsi que des apparitions aléatoires à l'écran.

J'ai également créé des scripts annexes (localisation_pixel.py, modifier_image.py) pour repérer précisément la position des indices dans les images et préparer les assets graphiques.

Mon rôle

Ce projet a été réalisé de manière individuelle : conception du scénario, développement du gameplay en Python/Pygame, gestion des assets (images, polices, musique) et mise en place de l'ambiance sonore et visuelle.

Résultats

Le résultat est un petit jeu jouable de bout en bout : une introduction, trois scènes explorables, une mécanique de recherche d'indices, un écran de code avec minuteur, ainsi qu'un écran de victoire et un écran de game over. Ce projet m'a permis de découvrir le développement de jeux vidéo avec Pygame et de travailler sur la gestion d'assets multimédias (images, polices, musique).
